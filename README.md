# pcs-staking-frontend
A TypeScript frontend for managing PCS staking, because who doesn't love a good yield farm?

## What it does
This project provides a simple web interface for users to stake, unstake, and claim their rewards on the PCS platform. It's built with TypeScript, React, and a dash of web3 magic.

## Getting started
To get up and running, follow these steps:

1. Clone the repo: `git clone https://github.com/your-username/pcs-staking-frontend.git`
2. Install dependencies: `npm install` or `yarn install`
3. Start the app: `npm start` or `yarn start`

## Example usage
Once the app is running, you can stake some tokens by going to `http://localhost:3000` in your browser. Click the "Stake" button, select the token you want to stake, and confirm the transaction. That's it!
